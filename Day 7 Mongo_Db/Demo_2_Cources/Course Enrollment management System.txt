Course Enrollment Management System using MongoDB
Objective
To design and implement a MongoDB-based system for managing student enrollments, instructors, and courses in a university setup.
The demo will be performed using MongoDB Compass (GUI) and optionally verified via Mongo Shell or VS Code MongoDB extension.
